# TDNET適時開示一覧 UAT チェックリスト

## 確認画面・URL

| 環境 | URL |
|---|---|
| 本番 | `https://mini-tools-rho.vercel.app/tools/tdnet-disclosures` |
| Preview | Vercel PR コメントの URL + `/tools/tdnet-disclosures` |
| ローカル | `http://localhost:3000/tools/tdnet-disclosures` |

## データ取得の仕組み

| 表示 | 取得元 | 期待挙動 |
|---|---|---|
| 初期表示 | `MARKET_INFO_API_BASE_URL/tdnet/disclosures/latest` | latest の対象日を表示する |
| 日付指定 | `MARKET_INFO_API_BASE_URL/tdnet/disclosures/{date}` | 指定日の開示一覧を表示する |

## 正常系チェックポイント

- [ ] ページが正常表示される
- [ ] 初期表示で latest の対象日と総件数が表示される
- [ ] latest が休日で0件の場合、エラーではなく空データとして表示される
- [ ] 日付入力で `2026-05-15` を指定すると当日の一覧に切り替わる
- [ ] 日付の左右矢印で前日・翌日の一覧に切り替わる
- [ ] 検索範囲を「過去7日」にすると基準日を含む複数日分の開示が検索対象になる
- [ ] 検索範囲を「過去30日」にしてもページがエラーにならず、対象日数と総件数が表示される
- [ ] 複数日表示では各行の時刻欄に開示日が表示される
- [ ] 検索で銘柄コード、会社名、タイトルの一致項目だけに絞り込まれる
- [ ] 「財務関連のみ」で `is_financial_related=true` の項目だけが表示される
- [ ] 「決算短信のみ」で `is_earnings_release=true` の項目だけが表示される
- [ ] 「訂正を除外」で `is_correction=true` の項目が除外される
- [ ] カテゴリ選択で指定カテゴリの項目だけが表示される
- [ ] PDF / HTML / XBRL のリンク種別フィルタが動作する
- [ ] 「業績」で決算短信、業績予想、業績修正などの開示に絞り込まれる
- [ ] 「配当」で配当、剰余金関連の開示に絞り込まれる
- [ ] 午前・昼休み・午後・15:30以降の時間帯フィルタが動作する
- [ ] 条件クリアで検索文字列と全フィルタが初期化される
- [ ] PDF / HTML / XBRL リンクが存在する項目は別タブで開ける
- [ ] 銘柄コードリンクから Yahoo ファイナンスの日本株ページを別タブで開ける
- [ ] 横幅が狭い画面ではテーブルが横スクロールでき、文字が重ならない

## 異常系チェックポイント

| シナリオ | 期待する挙動 |
|---|---|
| `MARKET_INFO_API_BASE_URL` 未設定 | 取得失敗メッセージを表示する |
| API が 4xx / 5xx を返す | raw エラーを見せず、取得失敗メッセージを表示する |
| API が 5 秒でタイムアウト | 取得失敗メッセージを表示する |
| 存在しない日付を指定 | API の結果に応じてデータなしまたは取得失敗として表示する |
| 範囲指定中に一部の日付取得が失敗 | 取得できた日付分だけを結合して表示する |
