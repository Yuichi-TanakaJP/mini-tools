# ペンギンシューター 通常ステージ障害物システム

## 結論

通常小ステージに、破壊可能 / 破壊不可能の2種類の障害物を追加する。
迷路ステージは面白い拡張案として残しつつ、まずは敵と同時に出る通常ステージ障害物を先に実装する。

## 背景

ユーザーから、各ステージにランダム障害物を設置し、ぶつかるとライフが減る仕組みの提案があった。
また、敵が出ないステージでは障害物を迷路状に配置し、ゴール到達でクリアする案も出た。

## 決めたこと

- 通常小ステージでは障害物をランダムに出現させる
- 敵と同時に出る障害物は、下端側で避けられるように上から流れる形にする
- 破壊可能な障害物はショットで壊せる
- 破壊不可能な障害物はショットを吸収し、壊れない
- 障害物にぶつかるとライフが1減る
- ボムは破壊可能な障害物を壊せる
- エイリアンが出ない迷路ステージは次段階の拡張候補として扱う

## 理由

- 100小ステージ構成に、避ける / 壊す判断を追加できる
- まず通常ステージに障害物を入れることで、既存ゲームループへの影響を小さく確認できる
- 迷路ステージはクリア条件と敵出現ルールが変わるため、別PRで扱う方が安全
