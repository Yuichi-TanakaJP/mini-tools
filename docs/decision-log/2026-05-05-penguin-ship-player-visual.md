# 2026-05-05 ペンギン自機の宇宙船化

## 背景

- ペンギン・バニーシューターで、ペンギン単体がビームを出しているように見える点にフィードバックがあった
- ゲーム面の舞台は宇宙なので、自機表現も宇宙設定に合わせる必要があった

## 今回決めたこと

- 自機はペンギン単体ではなく、宇宙船に乗ったペンギンとして表示する
- 発射演出はペンギン本体からのビームではなく、宇宙船の船首ショットとして扱う
- 既存方針どおり外部画像アセットは追加せず、絵文字と CSS で表現する

## 判断理由

- 宇宙船に乗せることで、宇宙ステージと射撃アクションの文脈が自然につながる
- 外部画像を追加しない方針を保つことで、初回導入時の軽量な管理方針を維持できる
- ゲームルールや操作体系は変えず、違和感のある見た目だけを改善できる

## 影響範囲

- `app/tools/penguin-rabbit-shooter/ToolClient.tsx`
- `app/tools/penguin-rabbit-shooter/page.tsx`
- `app/page.tsx`

## 残課題

- 敵キャラクターの宇宙設定をさらに強めるかは未決定
- 効果音や専用画像素材の導入は今回も対象外
