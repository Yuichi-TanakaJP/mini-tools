# ペンギンシューター 背景・ボス演出強化

## 結論

PR9では、2人プレイと進捗保存には進まず、既存のゲーム画面を豪華版トライアルらしく見せるための背景・ボス演出を先に強化する。

## 背景

ペンギンシューターは、豪華にもりもりの計画で、これまで使っていない技術を試す位置づけになった。
一方で、PR8の2人プレイと進捗保存はユーザー判断で凍結中のため、次の実装対象から外す必要がある。

## 決めたこと

- 既存のステージ背景画像を残し、React側で多層背景を重ねる
- 大ステージIDごとに、空気感、遠景、中景、手前レイヤーを切り替える
- ボスは名前と攻撃ラベルに加えて、オーラ、危険リング、HPバーを表示する
- ボスHPバーは現在HPと最大HPから算出する
- PR8の2人プレイと進捗保存は引き続き凍結し、このPRでは触らない

## 理由

- 画像ファイルを大量追加する前に、ゲーム画面の情報量と演出密度を上げられる
- 町、国、月、火星、異次元の差分が、背景画像の読み込み状態に依存しすぎない
- ボスHPが見えることで、ボス戦の手応えをプレイヤーが理解しやすくなる
