# My Stocks Ratio Chart Types

Date: 2026-06-14

## Decision

`my-stocks` の `保有メモ` → `比率` 表示に、グラフ種別の切り替えトグル（`帯グラフ` / `円グラフ` / `四角ヒートマップ`）を追加する。

- `帯グラフ`: 従来の積み上げ帯（`StackedShareBar`）。
- `円グラフ`: SVG のドーナツ（`DonutChart`）。`stroke-dasharray` で各区分を 1 周 100 に正規化して描画する。
- `四角ヒートマップ`: squarified treemap（`TreemapChart` + `squarifyTreemap`）。各四角形のアスペクト比が 1 に近づくよう領域分割し、面積で比率を表す。

口座別・銘柄別のどちらのパネルも同じトグルで表示が切り替わる。各グラフの下には正確な %/金額を示す凡例（`ShareLegend` → `ShareRow`）を常に表示する。

銘柄別は上位 7 銘柄を個別表示し、それ以外は `他 N 銘柄` に集約して合計が全体と一致するようにする。

## Reason

取得額比率は口座区分・銘柄数が増えると帯グラフだけでは内訳を直感的に掴みにくい。円グラフは構成比、ヒートマップは銘柄数が多いときの面積比較に向くため、同じデータを複数の見せ方で切り替えられるようにした。

ライブラリは追加せず、SVG と CSS（絶対配置 + `aspect-ratio`）だけで描画する。端末内データのみを扱う既存方針・依存の軽さを維持するため。

## Impact

- `比率` タブは数量・取得単価が入力済みの保有銘柄を対象に、3 種類のグラフで比率を表示できる。
- 凡例は全グラフ共通で %/金額を表示するため、グラフ種別に依存せず正確な値を確認できる。
- 銘柄別の集計に `他 N 銘柄` バケットを導入し、上位 7 銘柄に入らない分も比率に反映される（従来は上位 8 件のみ表示で残りは無視していた）。
