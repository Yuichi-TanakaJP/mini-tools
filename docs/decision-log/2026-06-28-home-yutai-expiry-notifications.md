# 2026-06-28 ホーム通知に優待期限を追加

## 背景

- 株主優待期限帳には期限7日前の強調表示があるが、期限帳を開かないと気づけない。
- ホーム通知に市場イベントを集約したため、利用者自身の期限情報も同じ場所で確認したい要望が出た。

## 今回決めたこと

- ホーム通知に、今日から7日以内に期限を迎える優待を表示する。
- 対象は未使用かつ未アーカイブで、期限が設定されているアイテムに限る。
- 当日は「本日期限」、翌日以降は「あとN日」と表示し、期限が近い順に最大3件をプレビューする。
- 期限切れアイテムはホーム通知に残さない。
- 期限帳のLocalStorageをクライアントで直接参照し、登録内容をサーバーへ送信しない。
- 通知から株主優待期限帳へ移動できる導線を置く。

## 判断理由

- 7日以内は期限帳の既存の注意表示と揃い、通知過多を抑えながら使い忘れを防げる。
- 使用済み・アーカイブ済み・期限切れを除外すると、ホームには今すぐ行動できる項目だけが残る。
- 既存ストアを購読することで、別タブや期限帳での変更をホーム通知へ反映できる。
- LocalStorage内で完結させることで、既存の端末内保存方針を維持できる。
