# 2026-05-09 ペンギンシューター豪華版・新技術トライアル方針

## 背景

- ペンギンシューター初期版を `/tools/penguin-shooter` として追加した
- 初期版は CSS / 絵文字中心の軽量実装だったが、今後の方針として「豪華にもりもりの計画」として育てる意図が確認された
- このゲームは、これまで `mini-tools` で使っていない表現技術やゲーム実装技術を試すトライアル枠として扱う

## 今回決めたこと

- ペンギンシューターは、画像、BGM、効果音を積極的に導入する豪華版ゲームとして進める
- 5つの大ステージを持つ構成にする
  - 町
  - 国
  - 月
  - 火星
  - 異次元
- 各大ステージは20の小ステージを含み、合計100小ステージ構成にする
- 各大ステージには中間ボスとステージボスを置く
- 異次元ステージをクリアすると Shoot を救出し、世界が平和になるストーリーにする
- 10小ステージクリアで2人プレイを解放する
- ライフ最大値は10にする

## 判断理由

- 既存の `mini-tools` は投資系ツールや軽量ツールが中心で、ゲーム表現の試行余地が大きい
- ペンギンシューターを実験枠にすることで、画像アセット、音、演出、ゲーム進行、保存データなどを段階的に試せる
- 100小ステージ構成にすると、短期の達成感と長期の拡張余地を両立しやすい
- 大ステージごとに風景を分けることで、画像・BGM・敵・ボスの差分を自然に導入できる

- 2人プレイの操作方法を決める
